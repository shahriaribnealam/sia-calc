# SIA Calc 🧮

SIA Calc is a powerful, web-based scientific calculator designed for fast and accurate calculations. It features a modern, clean, and user-friendly interface to solve everything from basic arithmetic to complex scientific equations.

🌐 **Live Demo:** [https://sia-calc.pages.dev](https://sia-calc.pages.dev)

---

## ✨ Features

- **Basic & Advanced Math:** Easily perform addition, subtraction, multiplication, division, and percentage calculations.
- **Trigonometry:** Support for sine, cosine, tangent, and their inverses ($sin$, $cos$, $tan$, $sin^{-1}$, etc.).
- **Logarithms & Exponents:** Calculate natural logs ($ln$), common logs ($log$), square roots ($\sqrt{x}$), and powers ($x^y$).
- **Memory Functions:** Standard calculator memory features ($M+$, $M-$, $MR$, $MC$).
- **Responsive Design:** Optimized for all devices, including desktops, tablets, and smartphones.
- **Fast & Accurate:** Built with precise mathematical logic for instant results.
